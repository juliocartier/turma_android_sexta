# Trabalho Prático: Programação para Dispositivos Moveis em Android para Impacto Social

**Disciplina:** PROGRAMAÇÃO PARA DISPOSITIVOS MÓVEIS EM ANDROID

**Valor**: 0 a 10 pontos (Peso: 70% Projeto Técnico | 30% Apresentação)


## Objetivo

O objetivo deste trabalho é desenvolver um aplicativo funcional utilizando React Native que resolva um problema real ou traga benefícios para a sociedade. O projeto deve integrar interface, lógica de programação e persistência de dados.

## Temas Sugeridos

Os alunos podem escolher um dos temas abaixo ou propor um novo, desde que focado em impacto social: 

* **Solidariedade:** Plataforma para doação de alimentos, roupas ou móveis.  
* **Cidadania:** Registro de problemas urbanos (iluminação, buracos, focos de dengue).  
* **Apoio Local:** Guia de serviços de pequenos empreendedores e autônomos do bairro.
* **Saúde/Bem-estar:** Controle de medicação para idosos ou diário de sintomas comunitários.
* **Educação:** Aplicativo de troca de livros ou monitoria entre alunos.

**Obs: pode ser outro tema além desses acima.**

## Requisitos Técnicos

Para que o trabalho seja considerado completo, o aplicativo deve conter:

**Obs: O foco da disciplina é utilizar o framework React Native para construir a solução de forma híbrida.**  

1. **Framework:** React Native (CLI ou Expo).

2. **Navegação:** Uso de React Navigation (Stack, Tab ou Drawer).

3. **Persistência de Dados:** Implementação obrigatória de banco de dados (escolha uma opção):

    * **Firebase (Firestore):** Para soluções que exijam sincronização em nuvem.

    * **Dexie.js / SQLite:** Para soluções focadas em privacidade e funcionamento offline.

4. Operações CRUD: O app deve permitir, no mínimo, Criar, Ler, Atualizar e Deletar registros.

5. Interface (UI/UX): Layout responsivo e focado na usabilidade do público-alvo.
